HTML Tags

HTML tags are like keywords which defines that how web browser will format and display the content.

Common HTML Tags

Tag Name Description
<html> HTML Represents the root (top-level element) of an HTML document.
<head> Head Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
<title> Title Defines the document's title that is shown in a browser's title bar or a page's tab.
<body> Body Represents the content of an HTML document.
<h1...h6> Heading 1 through 6 Represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest.
<img> Image Embeds an image into the document.
<ul> & <ol> Unordered List and Ordered List Represents an unordered and ordered list of items â€” typically rendered as a numbered list while the latter is as a bulleted list.
<li> List Item Represents an item in a list.
<p> Paragraph Represents a paragraph.
<a> Anchor Creates a hyperlink.
<div> Division The generic container for flow content.
<span> Span A generic inline container for phrasing content.

HTML Table Tags

Tag Name Description
<table> Table Represents tabular data.
<tbody> Table Body Encapsulates a set of table rows.
<thead> Table Header Defines a set of rows defining the head of the columns of the table.
<tfoot> Table Footer Defines a set of rows summarizing the columns of the table.
<td> Table Data Defines a cell of a table that contains data.
<th> Table Head Defines a cell as a header of a group of table cells.
<tr> Table Row Defines a row of cells in a table.

Semantic HTML Tags

Tag Name Description
<article> Article Represents a self-contained composition in a document, page, application, or site.
<aside> Aside Represents a portion of a document whose content is only indirectly related to the document's main content.
<footer> Footer Represents a footer.
<header> Header Represents introductory content.
<nav> Navigation Represents a section of a page whose purpose is to provide navigation links.
<main> Main Represents the dominant content of the body of a document.
<section> Section Represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it.
<figure> Figure Represents self-contained content, potentially with an optional caption.
<figcaption> Figure Caption Represents a caption or legend describing the rest of the contents of its parent <figure> element.
<video> Video Embeds a media player which supports video playback into the document.
<audio> Audio Used to embed sound content in documents.

HTML Attributes

Common HTML Attributes

Attribute Belongs To Description
class Global Attribute Often used with CSS to style elements with common properties.
id Global Attribute Defines a unique identifier (ID) which must be unique in the whole document.
style Global Attribute Contains CSS styling declarations to be applied to the element.
alt <area>, <img>, <input> Alternative text in case an image can't be displayed.
colspan <td>, <th> The colspan attribute defines the number of columns a cell should span.
rowspan <td>, <th> Defines the number of rows a table cell should span over.
target <a>, <area>, <base>, <form> Specifies where to open the linked document.
href <a>, <area>, <base>, <link> The URL of a linked resource.
rel <a>, <area>, <link> Specifies the relationship of the target object to the link object.
src <audio>, <embed>, <img>, <video> The URL of the embeddable content.
